Laravel 11 Initial Setup Guide

1. Install Composer

Composer is a dependency manager for PHP. Ensure you have Composer installed on your machine. You can download and install Composer from the official website:

https://getcomposer.org/

2. Create a New Laravel Project

Use Composer to create a new Laravel 11 project. Open your terminal and run the following command:

composer create-project --prefer-dist laravel/laravel:^11.0 my-laravel-app

This will create a new Laravel 11 project in a directory named my-laravel-app.

3. Navigate to the Project Directory

Once the project is created, navigate to the project directory using the following command:

cd my-laravel-app

4. Serve the Application

You can serve the application locally using Laravel's built-in development server. Run the following command:

php artisan serve

By default, your application will be accessible at http://localhost:8000.

5. Configure Environment Variables

Laravel uses environment variables to configure settings. The environment file is located at:

.env

You can set up your database connection and other settings here.

6. Database Migration

Run the following command to create the database tables defined in your migrations:

php artisan migrate
